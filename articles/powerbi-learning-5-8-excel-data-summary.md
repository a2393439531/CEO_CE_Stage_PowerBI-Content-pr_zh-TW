# Excel 中 Power BI 摘要資料

在集合中的主題，您學到運用您現有的 Excel 活頁簿檔案上傳及探索它們在 Power BI 中是多麼的容易。 簡單的資料表的 Excel 活頁簿或建立 Excel 的更進階 BI 功能如 Powerpivot 和 Power View 的活頁簿，您可以上傳。
