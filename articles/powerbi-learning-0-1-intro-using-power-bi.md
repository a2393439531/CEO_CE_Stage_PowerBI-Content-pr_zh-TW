將說明您可以使用 Power BI 做許多項目，請記住所有這些活動，以及如何使用 Power BI 的所有分析通常依照一般流程。  **一般流程** 的 Power BI 中的活動，如下所示︰

-   將資料送回 Power BI Desktop，並建立報表。
-   發行至 Power BI 服務，您建立新的視覺效果或組建儀表板
-   與其他人，特別是出門的人共用儀表板
-   檢視並與共用的儀表板和報表互動 Power BI 行動應用程式中
