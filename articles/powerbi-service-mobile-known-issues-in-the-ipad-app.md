# IPad 應用程式 (適用於 iOS 的 Power BI) 中的已知的問題  

## 「 我們發生通訊失敗 」  
您可能會看到此訊息，如果您的 iPad 網際網路連線需要強制明確 HTTP proxy 設定，手動或自動。 在此情況下，Power BI 應用程式將無法運作，您的 ipad。

### 因應措施  
切換至另一個連接，而不需要明確 HTTP proxy 設定 （亦即，一個設定為已關閉的 HTTP Proxy） iPad。
